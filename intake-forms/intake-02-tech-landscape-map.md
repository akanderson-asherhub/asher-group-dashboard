# Tech Landscape Map (Quarterly Briefing) Intake Form

## Engagement: Tech Landscape Map (Quarterly)
**Duration:** 2 weeks per update | **Fee Range:** $1.5K–$5K per quarter

---

## Required Fields

| Field | Type | Description |
|---|---|---|
| Client Name/Fund | text | PE/CVC fund, Owner-operator, or GC name |
| Client Sector Focus | select | GC / Owner-Operator / Subcontractor / Vendor / PE/CVC / Other |
| Technology Categories of Interest | multi-select | Project Management / Workforce Management / Financials / Safety / Equipment / Supply Chain / Tenant Management / Facilities Mgmt / Market Analytics / Other |
| Specific Competitors to Track | text | Comma-separated vendor names (e.g., "Bridgit, Procore, Touchplan") |
| Geography | select | US Only / North America / Europe / Asia-Pacific / Global |
| Use Case for Intelligence | select | Competitive positioning / Investment due diligence / Customer benchmarking / M&A target scouting / Internal strategy |
| Reporting Frequency | select | Monthly / Quarterly (standard) / Ad hoc |

---

## Optional Enrichment Fields

| Field | Type | Notes |
|---|---|---|
| Fund Size (if PE/CVC) | text | e.g., "$500M" or "Mega-fund $2B+" |
| Investment Stage Preference | text | Seed / Series A / Series B / Growth / All |
| Key Themes | text | E.g., "Labor automation," "AI in construction," "ESG data" |
| Contact for Updates | text | Name, email, phone for quarterly briefings |
| Budget/Deal Pipeline | text | If applicable, provide context for relevance (e.g., "evaluating PropTech in Q2") |

## Auto-Population Notes

- **Client Name, Sector, Categories:** Pre-fill if returning client

## Quarterly Deliverable Includes

- **Competitive positioning update:** New vendors, funding, acquisitions, exits
- **Deal flow summary:** Notable funding rounds, acquirers, market consolidation
- **Emerging trends:** Labor shortage, regulatory, ESG, AI adoption in focus categories
- **Portfolio company benchmarking:** How do client's portfolio companies compare to market?
- **Brief (1–2 page) + Data (XLSX of new vendors/funding)**
