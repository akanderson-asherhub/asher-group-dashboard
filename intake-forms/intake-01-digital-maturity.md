# Digital Maturity Assessment Intake Form

## Engagement: Digital Maturity Assessment
**Duration:** 4–6 weeks | **Fee Range:** $5K–$10K

---

## Required Fields

| Field | Type | Description |
|---|---|---|
| Company Name | text | Legal entity name |
| Trade/Sector | select | GC / Subcontractor / Specialty Contractor / Owner-Operator / Vendor |
| Annual Revenue (USD) | text | e.g., "$50M–$100M" or "$250M+" |
| Employee Headcount | number | Approximate total employees |
| Primary Markets | text | Geographic focus (e.g., "Northeast, Mid-Atlantic") |
| Current Tools List | text | Comma-separated: Project mgmt (Procore/Bridgit), Financials, HR, Equipment, Other |
| Biggest Pain Point | select | Data silos / Manual workflows / Lack of visibility / Integration failures / High implementation costs / Other |
| Assessment Urgency | select | Immediate (next 30 days) / Soon (next 60 days) / Planning phase (3–6 months) |

---

## Optional Enrichment Fields

| Field | Type | Notes |
|---|---|---|
| Executive Sponsor | text | Name, title, email (if known) |
| Number of Project Sites | number | Active project count or average |
| IT Budget (Annual) | text | If available, helps contextualize investment capacity |
| Prior Assessments | text | Have you done a tech audit before? When? With whom? |
| Specific Business Drivers | text | E.g., "scaling to 10 regions," "labor shortage," "ESG compliance" |

## Follow-Up Questions (Post-Submission)

Upon receipt, Asher Group follows up within 24 hours to confirm:

- Confidentiality / non-disclosure agreement if required
