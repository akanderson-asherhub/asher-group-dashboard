# Technology Roadmap Intake Form

## Engagement: Technology Roadmap
**Duration:** 8–12 weeks | **Fee Range:** $25K–$75K

---

## Required Fields

| Field | Type | Description |
|---|---|---|
| Company Name | text | Legal entity name |
| Annual Revenue | text | e.g., "$50M–$100M" or "$500M+" |
| Trade/Sector | select | GC / Subcontractor / Specialty / Owner-Operator / Vendor / Other |
| Current Maturity Score | number | 1–5 scale (or "Unknown, to be assessed") |
| Strategic Priorities | multi-select | Labor efficiency / Cost visibility / Safety & compliance / Client satisfaction / Data & analytics / Integration / Scalability / Other |
| Budget Range (3-year) | text | e.g., "$200K–$500K" or "TBD pending business case" |
| Key Stakeholders | text | Names/titles: CEO, COO, IT Director, Operations VP, CFO |
| Urgency | select | This quarter / Next 6 months / Next 12 months / Planning phase |

---

## Optional Enrichment Fields

| Field | Type | Notes |
|---|---|---|
| Number of Project Sites | number | Geographic spread and scale |
| IT Team Size | number | How many people support technology internally? |
| Current Tech Stack Summary | text | If known, list key systems (Procore, QuickBooks, HR platform, etc.) |
| Recent M&A or Integration | text | Any recent acquisitions or system consolidations? Impacts roadmap |
| Specific Pain Points | text | Free-text description of biggest operational challenges |
| Leadership Change | text | Any new executives driving digital transformation? Helpful for sponsorship |
| Prior Initiatives | text | Have you attempted tech initiatives before? Success/failure? Lessons learned? |

## Auto-Population Notes

- **Company Name, Revenue, Trade, Sites:** Pre-fill if repeat client or if prior digital maturity assessment exists
- **Current Maturity Score:** Auto-fill from Tech Audit Sprint (if done recently)
- **Current Tech Stack:** Pre-populate from stack inventory (if available)
- **Key Stakeholders:** Pre-fill from prior engagement governance structure

---

## Pre-Engagement Meeting

After intake, Asher Group schedules 30-minute kickoff call to confirm:
- Steering committee sponsor and availability
- Scope boundaries (which business functions to assess?)
- Timeline and budget alignment
- Any confidentiality or cultural sensitivities

---

## Roadmap Deliverables Include

- **Current State Assessment:** Tech stack inventory, gap analysis, maturity scorecard
- **Market Analysis:** Vendor landscape, best-in-class benchmarks by category
- **Strategic Roadmap:** 3-year phased plan (Quick Wins, Near-term, Strategic initiatives)
- **Business Cases:** Cost-benefit analysis for each major initiative
- **Implementation Plan:** Vendor selection, project timeline, change management, success metrics
- **Risk Register:** Identified risks, mitigation strategies, contingencies
