# Engagement SOW Intake Form

## Engagement: Consulting SOW (All Engagement Types)
**Variable duration and fees based on engagement type**

---

## Required Fields

| Field | Type | Description |
|---|---|---|
| Client Name | text | Your company or organization name |
| Engagement Type | select | Digital Maturity / Tech Landscape Map / Tech Audit Sprint / Vendor Evaluation / Tech Roadmap / Market Entry Strategy / Business Case & ROI / Commercial DD / Investment Thesis / Advisory Retainer (Light) / Advisory Retainer (Active) / Fractional CINO |
| Scope Summary | text | 2–3 sentence description of what you're trying to accomplish |
| Key Deliverables Required | text | Comma-separated (e.g., "strategy roadmap, financial model, vendor scorecard, implementation plan") |
| Desired Timeline | text | e.g., "8 weeks" or "by end of Q2" |
| Budget / Fee Range | text | If you have a budget range, enter it; otherwise "To be quoted" |
| Primary Contact | text | Name, title, email, phone |
| Executive Sponsor | text | Name, title, email |

---

## Optional Enrichment Fields

| Field | Type | Notes |
|---|---|---|
| Confidentiality Level | select | Standard / Confidential / Highly Confidential (requires NDA) |
| Steering Committee Members | text | Names, titles of decision-makers |
| Special Requirements | text | Site visits needed? Multi-stakeholder interviews? Board reporting? |
| Budget Constraints | text | Any hard limits? Flexibility for scope trade-offs? |
| Go/No-Go Decision Date | date | When must recommendation be finalized? |
| Follow-On Engagement Planned | text | Will this lead to implementation work or advisory retainer? |

## Auto-Population Notes

- **Client Name, Primary Contact, Executive Sponsor:** Pre-fill if repeat client or warm introduction
- **Engagement Type:** If client mentions specific pain point, pre-suggest matching engagement type
- **Budget:** If client is repeat, reference past engagement fees as anchor

---

## Payment Structure (Standard 30/30/40)

Upon SOW signature, payments are structured as:
- **30%** upon kickoff and Phase 1 deliverables acceptance
- **30%** upon Phase 2 completion and steering committee sign-off
- **40%** upon final delivery acceptance

All invoices are due Net 15. Late payment subject to 1.5% monthly interest.

---

## Post-Intake Process

Upon receipt, Asher Group:
1. Confirms scope with client (30-minute call if needed)
2. Generates custom SOW based on engagement type and specific requirements
3. Sends SOW for signature within 48 hours
4. Schedules kickoff upon execution
