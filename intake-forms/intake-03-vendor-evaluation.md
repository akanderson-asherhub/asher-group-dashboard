# Vendor Evaluation Toolkit Intake Form

## Engagement: Vendor Evaluation Toolkit
**Duration:** 3–4 weeks | **Fee Range:** $8K–$15K

---

## Required Fields

| Field | Type | Description |
|---|---|---|
| Company Name | text | Your company name |
| Software Category Being Evaluated | select | Project Management / Workforce Management / Financials / Safety / Equipment / Supply Chain / Facilities Management / Other |
| Current System (if any) | text | Existing tool name (e.g., "Procore") or "None" |
| Must-Have Requirements | text | 5–7 non-negotiables (e.g., "Cloud-based, mobile app, subcontractor portal, real-time budget tracking") |
| Budget Range (3-year TCO) | text | e.g., "$50K–$100K" or "Open" |
| Decision Timeline | select | This month / Next month / Q2 / Q3 / TBD |
| Number of Users | number | Estimated active users at go-live |
| Integration Requirements | text | Systems to connect to (e.g., "Procore, QuickBooks, Autodesk, Salesforce") |

---

## Optional Enrichment Fields

| Field | Type | Notes |
|---|---|---|
| Vendor Shortlist | text | If known, comma-separated (e.g., "Touchplan, Bridgit, Bridger") |
| Trade/Sector | select | GC / Subcontractor / Specialty / Owner / Vendor |
| Number of Project Sites | number | Helps contextualize scale |
| Mobile vs. Desktop Priority | select | Mobile-first / Desktop-first / Both equally |
| Reporting Needs | text | Specific KPIs or dashboards required (e.g., "Real-time crew utilization, daily production tracking") |
| Change Management Concerns | text | Any specific organizational concerns (e.g., "Resistance from ops team," "Limited IT support") |

## Auto-Population Notes

- **Company Name, Trade, Project Sites:** Pre-fill if repeat client
- **Current System, Integration Requirements:** Auto-populate if known from prior assessments
- **Must-Have Requirements:** Help Asher Group rapidly create evaluation scorecard

---

## Deliverables Include

- **Requirements Matrix:** Structured mapping of must-haves to vendor capabilities
- **Vendor Scorecard:** Feature/cost/implementation comparison (3–4 vendors evaluated)
- **Financial Analysis:** TCO comparison (software + implementation + training costs)
- **Negotiation Playbook:** Pricing benchmarks, contract red-flags, negotiation tactics
- **Go/No-Go Recommendation:** Ranked vendor recommendation with rationale
