# Market Entry Strategy Intake Form

## Engagement: Market Entry Strategy
**Duration:** 10–14 weeks | **Fee Range:** $40K–$100K

---

## Required Fields

| Field | Type | Description |
|---|---|---|
| Company/Product Name | text | Your company or product name |
| Target Market Segment | text | GC / Subcontractor / Specialty / Owner-Operator / Vendor / Real Estate Pro / Other |
| Current Stage | select | Pre-product / MVP/Beta / Early revenue (<$500K ARR) / Growth ($500K–$5M ARR) |
| Target Geography | multi-select | US (specify region) / Canada / Europe / Other |
| Known Competitors | text | Comma-separated competitor names |
| Current/Planned Funding Stage | select | Bootstrapped / Seed / Series A / Series B+ / Other |
| Use Case | select | Internal implementation / Investor pitch / Partnership development / M&A target identification |

---

## Optional Enrichment Fields

| Field | Type | Notes |
|---|---|---|
| Founding Team Background | text | Prior experience, domain expertise, startup track record |
| Current Traction | text | Pilot customers, LOIs, waitlist size, early revenue |
| Biggest Market Uncertainty | text | What do you most need to validate? Buyer willingness? Pricing? Scalability? |
| Product Roadmap (12-month) | text | Major features planned; impacts go-to-market strategy |
| Sales Model | select | Direct sales / PLG (Product-Led Growth) / Channel / Marketplace / Hybrid |
| Pricing (if decided) | text | Per-seat / Per-project / SaaS subscription / Other; current pricing if any |
| LinkedIn Profile | text | URL to CEO/founder profile (helpful for research context) |

## Market Entry Strategy Deliverables Include

- **Market Sizing:** TAM/SAM/SOM for target segment, growth projections
- **Competitive Analysis:** Positioning vs. direct and indirect competitors, win/loss scenarios
- **Segment Selection:** Recommended initial target segment (e.g., "100–250 person GCs in West" vs. "National sub-100 person specialists")
- **Go-To-Market Playbook:** Sales channel, customer acquisition strategy, pricing, messaging
- **Customer Development Plan:** Ideal customer profile, interview research, validation tactics
- **12-Month Roadmap:** Product, sales, funding, partnership milestones
- **Pitch Deck & Investor Materials:** Refined framing for seed/Series A pitch
